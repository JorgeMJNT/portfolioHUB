Projetos

Nesta seção, apresento alguns projetos que representam minha atuação prática em tecnologia, ciência de dados e Inteligência Artificial.

Análise de Sentimento com NLP

Objetivo: Classificar automaticamente textos com base no sentimento expresso.

Descrição: Desenvolvimento de modelo com processamento de linguagem natural e machine learning para interpretar grandes volumes de texto e gerar insights analíticos.

Tecnologias: Python, NLP, Machine Learning, scikit-learn.

transcrever_offline

Objetivo: Automatizar a transcrição de vídeos em texto estruturado.

Descrição: Solução com Inteligência Artificial para extrair áudio, gerar transcrição automática e produzir legendas em formato SRT.

Tecnologias: Python, Faster-Whisper, MoviePy, SRT.

YouTubeLanguageDetector

Objetivo: Identificar automaticamente o idioma de conteúdos digitais.

Descrição: Pipeline de classificação de idiomas com NLP aplicado à análise de textos em escala.

Tecnologias: Python, NLP, classificação de texto.

Projetos em AWS

Objetivo: Explorar o uso de infraestrutura em nuvem em soluções modernas.

Descrição: Aplicação de conceitos de cloud computing para deploy, organização de aplicações e suporte à escalabilidade.

Tecnologias: AWS, cloud computing.

Sistema de Pedidos

Objetivo: Estruturar uma aplicação com arquitetura moderna e escalável.

Descrição: Desenvolvimento de sistema com APIs e microserviços, com foco em modularidade e organização dos componentes.

Tecnologias: C#, .NET, APIs REST, microserviços.
